Modulo Para Actualizacion de Aplicaciones

Metodo de Actualizacion

1.- Actualizar Aplicaciones

En la pestaña de aplicaciones se encontrara la opcion "Actualizar Aplicaciones" la cual al momento de seleccionar nos abrira el asistente que importara la actualicacion de el modulo que desea actualizar.

2.- Importar el Modulo

Al abrir el asistente para actualizar el modulo se mostrara la opcion en donde podras seleccionar el modulo actualizado que se importara a Odoo, sustituyendo al que ya se encuentra dentro del mismo.

3.- Buscar el Modulo Actualizado

Al momento de seleccionar "Sube tu archivo" se abrira el gestor de archivos en donde se buscara el modulo actualizado para que posteriormente se pueda importar. Nota: El modulo debera estar comprimido en formato "tar.gz" ya que de lo contrario no se podra importar y el nombre de la actualizacion debe ser el mismo que el modulo que se actualizara dentro de Odoo.

4.- Actualizar

Ya que se selecciono el modulo actualizado se mostrara el archivo seleccionado y solo queda seleccionar el boton "Actualizar" para que el modulo se sustituya por el que se encuentra dentro de Odoo.

Notas Adicionales

Como ya se menciono antes, al momento de actualizar un modulo la actualizacion que fue seleccionado sustituira por completo al modulo que se encontraba dentro del sistema y ya no se podra recuperar la informacion del modulo anterior.

Modificaciones Adicionales

Para que la Aplicacion funcione correctamente se tendran que hacer algunas modificaciones, una de ellas es que trabajaremos con la libreria screen asi que se tendra que instalar en nuestro ordenador. Para ello nos situaremos en nuestra consola y escribimos el siguiente comando:

sudo apt-get install screen

Ya que tenemos instalado el screem debemos crear una carpeta llamada "scripts", para ello dentro de nuestra consola nos posicionamos en el directorio raiz en donde escribiremos:

sudo mkdir scripts

Posteriormente creamos el Bash que estara dentro de la carpeta que acabamos de crear para eso escribimos en nuestra terminal:

cd $HOME && touch restart_odoo.sh && chmod +x restart_odoo.sh

cd $HOME && echo '#!/bin/bash' > restart_odoo.sh && echo '# -*- ENCODING: UTF-8 -*-' >> restart_odoo.sh

Ahora buscamos el Bash que acabamos de crear dentro de la carpeta "home" y lo movemos a la carpeta scripts, dentro del archivo nos colocamos en el directorio en donde se encuentra Odoo y añadimos los comandos para cerrar y abrir el sistema.

En este caso nuestro local en el que se encuenta nuestro sistema es el 8069 utilizamos el comando "fuser -k 8069/tcp" para cerrarlo y para abrirlo nuevamente sera con el comando "python odoo-bin -c odoo.conf".

Esto puede variar dependiendo la manera de abrir Odoo de cada uno y el local en donde se encuentre

Esto servira para que, al momento de actualizar un modulo, el sistema pueda reconocer los cambios hechos en el modulo actualizado.
